# Dwm configuration of Yuyu's 

![](dwm.png)

## Installation

* Clone the repo to wherever you want, for me,  `~/.config/dwm` would be good.

```
$ git clone https://github.com/yuyuranium/dwm.git
```

* Make it.

```
$ cd dwm
$ sudo make clean install
```

* Restart your display server
